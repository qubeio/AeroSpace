// end::synopsis[]

// =========================================================== Description
== Description

// tag::body[]
{manpurpose}

Reinserts tiled windows in their existing depth-first order using BSP insertion. Floating windows are
left unchanged. Use this command to recover a workspace whose root was previously changed to `tiles`
or `accordion`.
